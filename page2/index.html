Recover Lost QuickTime Recordings

06 Jun 2014

Step 1

Reopen QuickTime Player. Your recording might still be avilable as an unsaved file. If step 1 fails, go to step 2.

Step 2

  1. Go to ~/Library/Containers/com.apple.QuickTimePlayerX/Data/Library/Autosave Information
  2. You should be able to see a file named Unsaved QuickTime Player Document.qtpxcomposition > Right Click > Show Package Contents
  3. The unsaved recording will be named Audio Recording.m4a

ExpressCard USB on MacBook Pro Late 2008

25 Apr 2014

Notice how the original USB ports are all used up. You can add USB ports with an ExpressCard USB. You can get it cheaply on eBay or Amazon for around $10.

My MacBook Pro (Late 2008) does not readily support the ExpressCard USB. You have to download 3rd party drivers.

Requirements:

  1. Make sure you get an Expresscard USB that has a uPD720202 Chipset.
  2. Download the driver by zenith432
  3. Download Kext Utility.
  4. Install the driver using Kext Utility and restart.

Some Problems:

  1. Most USB devices will work. However, a mouse and USB external drive have failed so far.
  2. The Expresscard USB may get stuck in the MacBook Pro.

To remove the Expresscard menubar icon, go to /System/Library/CoreServices/Menu Extras and rename "ExpressCard.menu". Restart again.

More help/info can be found at insanelymac.com

Turn App Nap Off (Script)

10 Mar 2014

Since there’s no way to turn app nap off universally, I’ve created a script that turns on/off app nap for applications in the Applications folder (including apps in subfolders).

How To Use

  1. System requirements: OSX Mavericks

  2. Download the zip file and extract it.

  3. Double click the Automator workflow > click “Install” > click “Done”.

  4. Run the service from Finder.

  5. Alternatively, you can just open the workflow in Automator and run it from there.

    Note: Use this at your own risk.
